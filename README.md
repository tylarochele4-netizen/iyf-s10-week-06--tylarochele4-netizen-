# Week 6: Asynchronous JavaScript - Weather Dashboard

## 🚀 Projects Included

### 1. Weather Dashboard (Mini-Project)
A real-time weather application that fetches data from the OpenWeatherMap API.
* **Features:** Search by city, dynamic UI updates based on weather conditions, and error handling for invalid city names.
* **Tech Used:** Fetch API, Async/Await, DOM Manipulation.

### 2. User Directory
A dynamic directory that fetches placeholder user data.
* **Features:** Live search filtering and data rendering using the JSONPlaceholder API.

### 3. Async Exercises
Refactored code demonstrating the transition from:
* **Callbacks** → **Promises** → **Async/Await**

---

## 🛠️ Concepts Learned

* **The Event Loop:** Understanding how JavaScript handles non-blocking operations.
* **Promise Methods:** Using `Promise.all()` to handle multiple API requests simultaneously.
* **Error Handling:** Implementing `try...catch` blocks to provide a smooth user experience even when network requests fail.
* **API Integration:** Working with API Keys, query parameters, and JSON response parsing.
